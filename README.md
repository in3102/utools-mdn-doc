# uTools 帮助文档生成

文档来源 https://developer.mozilla.org/zh-CN/docs/Web/

## 项目初始化

```
npm i
```

## 文档生成

### JavaScript 文档生成

```
 npm run javascript
```

### Html 文档生成

```
 npm run html
```

### Web API/DOM 文档生成

```
 npm run api
```

### CSS 文档生成

```
 npm run css
```

### HTTP 文档生成

```
 npm run http
```
